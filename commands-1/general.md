# Info Commands

## User Commands

| Command | Description |
| :--- | :--- |
| `$server` | Shows Server Info |
| `$avatar` | Shows Avatar |
| `$membercount` | Shows Server Member Count |
| `$botinfo` | Shows Bot Information |
| `$ping` | Ping The Bot |
| `$stats` | Shows Strider Statistics \(Servers, Users, Status\)  |

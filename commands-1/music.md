# Music Commands

{% hint style="info" %}
m$ Is The Standard Prefix For The Music Commands, It Cannot Be Changed Yet.
{% endhint %}

| Commands | Description |
| :--- | :--- |
| `$play (Song)` | Play A Song |
| `$stop` | Stop A Song |
| `$skip` | Skip A Song |
| `$lyrics` | Show Current Song Lyrics |
| `$np` | Shows Now Playing Song |
| `$q` | Shows The Queue |
